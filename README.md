# maimaiDX Roll 歌网页

毕竟 maimaiDX 删了随机选歌功能，所以就自己写了个来用。

---

## 安装环境与构建

先安装 [node.js](https://nodejs.org)，然后进入文件夹，输入命令：

```bash
npm i
```

开发时：

```bash
gulp
```

构建：

```bash
gulp build
```

嗯，KPI++。
